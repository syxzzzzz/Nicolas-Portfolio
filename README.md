# Personal Portfolio Website

This is a modern, responsive personal portfolio website designed for an IT student. The website showcases the student's projects, provides information about their background, and includes a contact form for inquiries.

## Project Structure

The project is organized as follows:

```
portfolio-website
├── src
│   ├── index.html          # Main entry point for the portfolio website
│   ├── css
│   │   ├── main.css        # Main styles for layout, typography, and color scheme
│   │   ├── responsive.css   # Styles for responsive design
│   │   └── components.css   # Styles for specific components (buttons, cards, forms)
│   ├── js
│   │   ├── main.js         # Main JavaScript file for global functionality
│   │   ├── portfolio.js     # JavaScript for the projects section
│   │   └── contact.js       # JavaScript for the contact form functionality
│   └── assets
│       └── fonts           # Custom fonts used in the website
├── pages
│   ├── about.html          # HTML structure for the about section
│   ├── projects.html       # HTML structure for the projects section
│   └── contact.html        # HTML structure for the contact section
└── README.md               # Documentation for the project
```

## Features

- **Hero Banner**: A visually appealing introduction to the portfolio.
- **Projects Section**: Displays various projects with descriptions and technologies used.
- **About Section**: Provides background information about the student.
- **Contact Form**: Allows visitors to get in touch with the student.

## Setup Instructions

1. Clone the repository to your local machine.
2. Navigate to the `portfolio-website` directory.
3. Open `src/index.html` in a web browser to view the portfolio.
